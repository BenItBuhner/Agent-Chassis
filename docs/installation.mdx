## System Requirements

- **Python 3.12+**
- **uv** package manager ([Install uv](https://github.com/astral-sh/uv))
- **Node.js** (optional, for Node.js-based MCP servers)

## Step-by-Step Installation

### 1. Clone the Repository

```bash
git clone https://github.com/BenItBuhner/Agent-Chassis.git
cd agent-chassis
```

### 2. Install Dependencies

```bash
uv sync
```

This installs all Python dependencies including:
- FastAPI and Uvicorn (web framework)
- OpenAI SDK (LLM integration)
- MCP (Model Context Protocol) libraries
- Optional: Redis, SQLAlchemy, JWT libraries

### 3. Configure Environment

Create a `.env` file from the template:

```bash
cp .env.example .env
```

Edit `.env` with your configuration. See [Environment Variables](/configuration/environment-variables) for details.

**Minimum required configuration:**
```env
OPENAI_API_KEY=sk-...
OPENAI_MODEL=kimi-k2-thinking
```

### 4. Configure MCP Servers (Optional)

Edit `mcp_config.json` to add MCP servers:

```json
{
  "mcpServers": {
    "Context7": {
      "url": "https://mcp.context7.com/mcp",
      "transport": "streamable-http"
    },
    "memory": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-memory"]
    }
  }
}
```

See [MCP Servers Configuration](/configuration/mcp-servers) for more details.

### 5. Run the Server

**Development mode (with auto-reload):**
```bash
uv run uvicorn app.main:app --reload
```

**Production mode:**
```bash
uv run uvicorn app.main:app --host 0.0.0.0 --port 8000
```

Visit `http://localhost:8000/docs` for interactive API documentation.

## Docker Installation

### Using Docker Compose

```bash
docker-compose up --build
```

### Using Dockerfile

```bash
docker build -t agent-chassis .
docker run -p 8000:8000 --env-file .env agent-chassis
```

## Verification

### Health Check

Check the health endpoint:

```bash
curl http://localhost:8000/health
```

Expected response:
```json
{
  "status": "healthy",
  "persistence_enabled": false,
  "user_auth_enabled": false,
  "redis_connected": false,
  "database_connected": false
}
```

### Test Agent Endpoint

```bash
curl -X POST http://localhost:8000/api/v1/agent/completion \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [
      {"role": "user", "content": "Hello!"}
    ],
    "model": "kimi-k2-thinking"
  }'
```

## Optional: Enable Persistence

If you want server-side session persistence:

1. **Set up Redis:**
   ```env
   REDIS_URL=redis://localhost:6379/0
   ```

2. **Set up PostgreSQL:**
   ```env
   DATABASE_URL=postgresql+asyncpg://user:pass@localhost/agent_chassis
   ```

3. **Enable persistence:**
   ```env
   ENABLE_PERSISTENCE=true
   ```

See [Persistence Guide](/guides/persistence) for detailed setup.

## Optional: Enable User Authentication

If you want JWT-based user authentication:

1. **Set JWT secret:**
   ```env
   JWT_SECRET_KEY=your-secure-secret-key-min-32-chars
   ```

2. **Configure Google OAuth (optional):**
   ```env
   GOOGLE_CLIENT_ID=your-client-id
   GOOGLE_CLIENT_SECRET=your-client-secret
   ```

3. **Configure email service:**
   ```env
   SMTP_HOST=smtp.gmail.com
   SMTP_PORT=587
   SMTP_USER=your-email@gmail.com
   SMTP_PASSWORD=your-app-password
   EMAIL_FROM=noreply@yourdomain.com
   ```

4. **Enable user auth:**
   ```env
   ENABLE_USER_AUTH=true
   ```

See [Authentication Guide](/guides/authentication) for detailed setup.

## Troubleshooting

### Port Already in Use

If port 8000 is already in use:

```bash
uv run uvicorn app.main:app --port 8001
```

### OpenAI API Key Not Found

Make sure your `.env` file contains:
```env
OPENAI_API_KEY=sk-...
```

### MCP Server Connection Issues

Check your `mcp_config.json` syntax and ensure MCP servers are accessible. See [MCP Servers Configuration](/configuration/mcp-servers) for troubleshooting.

### Database Connection Issues

If using persistence, verify:
- Redis is running: `redis-cli ping` (should return `PONG`)
- PostgreSQL is accessible: `psql -h localhost -U user -d agent_chassis`
